# Zen Feeds

Daily Mindful Moments. A curated feed of high-quality wallpapers paired with short, zen-inspired essays.

## Features
- **Curated Images**: S-grade selection from Unsplash.
- **AI-Powered Content**: Essays generated by Google Gemini to match the mood of each image.
- **Minimalist Design**: Focused on typography and visual clarity.

## Automation
Feeds are automatically generated using the latest additions from the Zen Wallpapers project.
